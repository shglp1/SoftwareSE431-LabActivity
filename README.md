# Software Maintenance Lab: To-Do App

A simple To-Do list application for a Software Maintenance & Evolution lab activity.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run the Application**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

-   `src/api.js`: Mock API service simulating backend interactions.
-   `src/components/`: React components.
-   `docs/PM-brief.md`: The client complaint describing the issue you need to investigate.

## The Scenario

You have inherited this codebase. The client is reporting issues with adding tasks. Your goal is to investigate, reproduce, and fix the issue while ensuring the code is maintainable.
